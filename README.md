# react-youtube-demo
YouTube Explorer  using React, Redux, and Webpack 

## Demo: http://youtube.founderstack.io

### Install
```
npm install
```
### Run DEV Server with Hot Reloading
```
npm run dev
```

### Run ESLint
```
npm run lint
```

### Run Tests
```
npm run test
```

### Run Production Build
```
npm run build
```
